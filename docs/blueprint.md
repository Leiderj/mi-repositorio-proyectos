# **App Name**: PNF-I ProjectHub

## Core Features:

- Project Registration Form: Comprehensive form to register project title, general and specific objectives, development method, locality, contact person information, and project dates (start/end).
- Team Member Management: Ability to add, edit, and view personal information for all project team members.
- File Upload & Management: Securely upload and store project-related images and bibliography PDFs. Projects can have multiple associated files.
- Project Organization by 'Trayecto': Categorize and view projects based on their 'Trayecto' (I, II, III, IV) for easy navigation and management.
- Project Search & Filter: Search and filter projects by title, and locality (parroquia) for quick retrieval of specific project information.
- Dynamic Report Generation: Generate on-demand reports, sortable by project name and parish, providing structured summaries of project data.
- Objective and Methodology Refinement Tool: A generative AI tool that suggests improvements for project objectives or proposes suitable research methodologies based on the project description.

## Style Guidelines:

- Primary color: A vibrant yet professional blue (#2377DB), reflecting trust and academic rigor. (HSL: 210, 70%, 45%).
- Background color: A very light, desaturated blue (#F4F8FA), providing a clean and readable canvas for content. (HSL: 210, 15%, 97%).
- Accent color: A deep, innovative teal (#1B8989), used for interactive elements and highlights to provide strong contrast. (HSL: 180, 70%, 35%).
- Body and headline font: 'Inter', a modern grotesque sans-serif, chosen for its excellent readability and neutral, objective appearance, ideal for clear presentation of academic and administrative content.
- Utilize clean, crisp line icons that are easily recognizable for form actions, navigation, and file types (e.g., upload, save, PDF, image).
- Employ a clear, multi-column grid system for content display, especially for project lists and reports, ensuring efficient use of space and readability. Forms should be logically grouped with clear labels.
- Subtle, non-distracting micro-animations for form submissions, data loading indicators, and success messages, providing visual feedback to the user without slowing down interactions.
